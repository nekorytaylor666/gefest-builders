## Базовый HTML

Ваша задача - создать базовую структуру HTML-страницы с использованием следующих тегов: `<!DOCTYPE html>`, `<html>`, `<head>`, `<title>`, `<body>`.

<br />
## Требования:

```html
<!DOCTYPE html>
```

1. Объявите документ как HTML5, используя `<!DOCTYPE html>`.
2. Используйте тег `<html>` для обозначения корня документа.
3. Внутри тега `<html>` добавьте тег `<head>` для обозначения метаданных документа.
4. Внутри тега `<head>` добавьте тег `<title>` и установите заголовок вашей страницы.
5. Используйте тег `<body>` для обозначения основного содержимого вашей страницы.
