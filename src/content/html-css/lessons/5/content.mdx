**Внешние отступы** (margin) в CSS устанавливают пространство вокруг элемента, отделяя его от других элементов. Они могут быть заданы для каждой стороны элемента (верхней, правой, нижней и левой) или общим значением для всех сторон.
+++
Примеры:

1. Задание внешнего отступа для всех сторон элемента:

<CodeEditor disabled lang="CSS">
  {`
margin: 10px;
`}
</CodeEditor>

1. Задание разных внешних отступов для каждой стороны элемента:

<CodeEditor disabled lang="CSS">
  {`
margin-top: 10px;
margin-right: 20px;
margin-bottom: 15px;
margin-left: 5px;`}
</CodeEditor>

+++

**Внутренние отступы** (padding) в CSS устанавливают пространство вокруг содержимого
элемента, отделяя его от границы элемента. Они также могут быть заданы для каждой
стороны элемента или общим значением для всех сторон.

Примеры:

1. Задание внутреннего отступа для всех сторон элемента:

<CodeEditor disabled lang="CSS">
  {`
padding: 10px;
`}
</CodeEditor>

1. Задание разных внутренних отступов для каждой стороны элемента:

<CodeEditor disabled lang="CSS">
  {`
padding-top: 10px;
padding-right: 20px;
padding-bottom: 15px;
padding-left: 5px;
`}
</CodeEditor>
