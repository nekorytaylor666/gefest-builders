HTML-теги позволяют форматировать текст на веб-странице. Некоторые из наиболее распространенных тегов для форматирования текста включают:

- `<strong>` - выделяет текст полужирным шрифтом.
- `<em>` - выделяет текст курсивом.
- `<u>` - подчеркивает текст.
- `<s>` - зачеркивает текст.
- ` <h1>``<h6> ` - определяют заголовки разных уровней.
- `<p>` - создает абзац текста.
- `<blockquote>` - создает блок цитаты.
- `<code>` - отображает текст в моноширинном шрифте, обычно используется для отображения кода.
- `<pre>` - отображает предварительно форматированный текст с сохранением пробелов и переносов строк.
- `<i>` - выделяет текст курсивом.
- `<b>` - выделяет текст полужирным шрифтом.
  +++

Примеры:

<CodeEditor disabled lang="HTML">

Это только небольшой набор тегов для форматирования текста в HTML. С помощью этих тегов вы можете создавать разнообразные стили и эффекты на веб-странице.
