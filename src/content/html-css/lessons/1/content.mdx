## Структура HTML

HTML (HyperText Markup Language) - это язык разметки, используемый для создания веб-страниц. Он состоит из тегов, которые определяют структуру и содержимое веб-страницы. Каждая веб-страница должна иметь определенную структуру, которая состоит из следующих основных элементов:

```html
<!DOCTYPE html>
```

Это объявление типа документа и сообщает браузеру, какую версию HTML следует использовать для отображения веб-страницы.

```html
<html></html>
```

Элемент `<html>` является корневым элементом веб-страницы. Он содержит все другие элементы на странице.

```html
<head></head>
```

Элемент `<head>` содержит метаданные документа, такие как заголовок страницы, ссылки на стили CSS и скрипты JavaScript.

```html
<body></body>
```

Элемент `<body>` содержит основное содержимое веб-страницы, такое как текст, изображения, ссылки и другие элементы.

+++

## Теги и атрибуты

Теги HTML используются для определения различных элементов на веб-странице, таких как заголовки, параграфы, списки и изображения. Каждый тег имеет определенную функцию и может иметь атрибуты, которые предоставляют дополнительную информацию о теге.

Примеры некоторых часто используемых тегов и атрибутов:

### Примеры тегов:

- `<h1>` - Заголовок верхнего уровня
- `<p>` - Параграф
- `<a>` - Ссылка
- `<img>` - Изображение

+++

- Пример использования тегов:

Вот пример полного HTML-кода с `<head>` и `<body>`, в котором есть теги `<p>`, `<b>`, `<i>`, `<u>`:

<CodeEditor disabled lang="HTML">
{`
<!DOCTYPE html>
<html>
  <head>
    <title>Мой HTML-документ</title>
  </head>
  <body>
    <h1>Заголовок страницы</h1>

    <p>Это обычный абзац текста.</p>

    <p><b>Это абзац с жирным текстом.</b></p>

    <p><i>Это абзац с курсивным текстом.</i></p>

    <p>
      <u>Это абзац с подчеркнутым текстом.</u>

      <a href="https://www.google.kz" class="link">Ссылка на Google</a>
    </p>

  </body>
</html>
`}
</CodeEditor>
В этом примере у нас есть `<head>`, который содержит заголовок документа `<title>`. В `<body>` мы имеем заголовок верхнего уровня `<h1>` и несколько абзацев `<p>`. В одном из абзацев мы используем тег `<b>` для жирного текста, тег `<i>` для курсивного текста и тег `<u>` для подчеркнутого текста.

Это простой пример, но вы можете добавить больше элементов и стилей по своему усмотрению.

+++

### Примеры атрибутов:

- `href` - Определяет URL-адрес для ссылки
- `src` - Определяет источник изображения
- `class` - Определяет класс элемента для стилизации с помощью CSS
- `id` - Определяет уникальный идентификатор элемента

Пример использования атрибутов:

<CodeEditor disabled lang="HTML">
  {`
<a href="<https://www.example.com>" class="link">Ссылка на пример</a>
<img src="image.jpg" id="my-image" />
`}
</CodeEditor>

+++

## Вложенность тегов в HTML

Теги HTML могут быть вложены друг в друга, что позволяет создавать структуру и организовывать содержимое веб-страницы. Вложенность тегов означает, что один тег может содержать в себе другой тег.

Вот пример кода, демонстрирующий вложенность тегов `<p>`, `<i>`, `<b>`, `<img>` и `<a>`:

<CodeEditor disabled lang="HTML">
{`
<!DOCTYPE html>
<html>
  <head>
    <title>Мой HTML-документ</title>
  </head>
  <body>
    <p>Это абзац с <i>курсивным текстом</i> и <b>жирным текстом</b>.</p>

    <p>
      Это абзац с <a href="<https://www.example.com>">ссылкой на пример</a> и
      <img src="image.jpg" /> изображением.
    </p>

  </body>
</html>
`}
</CodeEditor>

+++

В примере выше у нас есть два абзаца `<p>`. В первом абзаце мы используем тег `<i>` для курсивного текста и тег `<b>` для жирного текста. Во втором абзаце мы используем тег `<a>` для создания ссылки и тег `<img>` для отображения изображения.

Обратите внимание, что теги могут быть вложены внутрь других тегов, создавая иерархию элементов на веб-странице.

Вы можете добавить больше вложенных тегов и создавать более сложную структуру веб-страницы в зависимости от ваших потребностей.
