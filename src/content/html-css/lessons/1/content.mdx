## Псевдоклассы и псевдоэлементы в CSS

Псевдоклассы и псевдоэлементы в CSS используются для стилизации определенных состояний элементов или для создания дополнительных элементов на веб-странице.

### Самые часто используемые псевдоклассы:

- \:hover - применяется, когда курсор мыши находится над элементом
- \:active - применяется, когда элемент активен (нажатие на кнопку, например)
- \:focus - применяется, когда элемент находится в фокусе
- \:visited - применяется к посещенным ссылкам
- \:nth-child(n) - применяет стили к элементу, который является n-ным потомком своего родителя

### Примеры использования псевдоклассов:

```txt
a:hover {
  color: blue;
}

button:active {
  background-color: red;
}

input:focus {
  border-color: green;
}

li:nth-child(odd) {
  background-color: lightgray;
}

```

### Самые часто используемые псевдоэлементы:

- \::before - создает псевдоэлемент перед содержимым выбранного элемента
- \::after - создает псевдоэлемент после содержимого выбранного элемента
- \::first-letter - применяется к первой букве внутри выбранного элемента
- \::first-line - применяется к первой строке внутри выбранного элемента
