CSS (Cascading Style Sheets) - это язык стилей, который используется для оформления
и визуального оформления веб-страниц. Он определяет, как элементы HTML должны быть
представлены на экране, включая различные свойства, такие как цвет текста, размер
шрифта, отступы, фоны и многое другое.
+++
CSS играет важную роль в веб-разработке, поскольку позволяет создавать привлекательный и профессиональный дизайн для веб-страниц. С помощью CSS вы можете контролировать внешний вид и макет вашего сайта, делая его более привлекательным и пользовательски дружелюбным.

Вот пример CSS-кода:

<CodeEditor disabled lang="CSS">
{
`
body {
background-color: #f2f2f2;
}

h1 {
font-size: 24px;
}

p {
color: #666666;
}
`
}

</CodeEditor>

В этом примере устанавливаются стили для `<body>`, `<h1>` и `<p>`. Например, фоновый цвет `body` установлен на `#f2f2f2`, размер шрифта `h1` на `24px`, а цвет текста `p` на `#666666`.

CSS-код позволяет определить различные свойства элементов HTML и настроить их внешний вид веб-страницы.
