HTML предоставляет возможность создания таблиц с помощью элементов `<table>`, `<tr>`, `<th>`, и `<td>`. Вот пример кода, иллюстрирующий создание таблицы:
+++

<CodeEditor disabled lang="HTML">
  {`
<table>
  <tr>
    <th>Заголовок 1</th>
    <th>Заголовок 2</th>
    <th>Заголовок 3</th>
  </tr>
  <tr>
    <td>Ячейка 1</td>
    <td>Ячейка 2</td>
    <td>Ячейка 3</td>
  </tr>
  <tr>
    <td>Ячейка 4</td>
    <td>Ячейка 5</td>
    <td>Ячейка 6</td>
  </tr>
</table>
`}
</CodeEditor>

В этом примере у нас есть таблица с 2 строками и 3 столбцами. Первая строка является заголовком таблицы и содержит элементы `<th>`, а остальные строки содержат данные и используют элементы `<td>`. Элементы `<th>` и `<td>` должны быть вложены в элемент `<tr>`, который представляет собой строку.

Таким образом, вы можете создавать таблицы в HTML, указывая заголовки и данные для каждой ячейки.
