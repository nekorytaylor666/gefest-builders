+++ ## Нумерованные списки

Нумерованные списки в HTML создаются с помощью тега `<ol>`. Каждый элемент списка обозначается тегом `<li>`. Пример **кода**:

<CodeEditor disabled lang="HTML">
  {`
<ol>
  <li>Первый пункт</li>
  <li>Второй пункт</li>
  <li>Третий пункт</li>
</ol>
`}
</CodeEditor>

+++
Дополнительно, вы можете использовать атрибуты `start` (цифры) и `type` (A, a, I, i, 1) в теге `<ol>` для настройки начального значения списка и типа нумерации соответственно.

Пример кода с атрибутом `start`:

<CodeEditor disabled lang="HTML">
  {`
<ol start="3">
  <li>Третий пункт</li>
  <li>Четвертый пункт</li>
  <li>Пятый пункт</li>
</ol>
`}
</CodeEditor>

+++
Пример кода с атрибутом `type`:

<CodeEditor disabled lang="HTML">
  {`
<ol type="A">
  <li>Первый пункт</li>
  <li>Второй пункт</li>
  <li>Третий пункт</li>
</ol>
`}
</CodeEditor>

+++

## Маркированные списки

Маркированные списки в HTML создаются с помощью тега `<ul>`. Каждый элемент списка также обозначается тегом `<li>`. Пример кода:

<CodeEditor disabled lang="HTML">
  {`
<ul>
  <li>Первый пункт</li>
  <li>Второй пункт</li>
  <li>Третий пункт</li>
</ul>
`}
</CodeEditor>

+++
Дополнительно, вы **можете** использовать атрибут `type` в теге `<ul>` для настройки вида маркеров. Возможные значения для атрибута `type`:

- `disc`: круглые маркеры (по умолчанию)
- `circle`: пустые круглые маркеры
- `square`: квадратные маркеры
- `none`: отсутствие маркеров
  +++

  Пример кода с атрибутом `type`:

<CodeEditor disabled lang="HTML">
  {`
<ul type="circle">
  <li>Первый пункт</li>
  <li>Второй пункт</li>
  <li>Третий пункт</li>
</ul>
`}
</CodeEditor>
